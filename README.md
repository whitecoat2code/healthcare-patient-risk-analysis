# Healthcare Patient Risk Analysis

## Overview
This project analyzes hospital patient data to identify high-risk individuals using simple clinical indicators (age and blood pressure). It demonstrates SQL and Python (Pandas) skills applied to healthcare data.

## Files
- `dataset.csv` — sample patient data
- `sql_queries.sql` — ready SQL queries (SELECT, GROUP BY, ORDER BY, LIMIT)
- `analysis.py` — Python script using Pandas to run the analysis
- `high_risk_patients.csv` — generated by `analysis.py`
- `insights.md` — short clinical insights and recommendations

## Key findings (summary)
- Patients older than 50 with BP > 140 show higher mortality in this dataset.
- Average BP increases with age decade.
- Top BP patients can be prioritized for intervention.
